TeleVision

A TV as a Game Overlay

Play in a Television

This program is allowing to have a custom television as an overlay for playing your game inside the screen. Let's see...

Considerations About the Program

You shall use a GIF file, it will be a television animated. The file is under assets folder where the program is. You must use keyboard key page down under your game in foreground. Use keyboard key shortcuts Alt + Tab to browse windows and Alt + F4 to close current window. To see it, your game, you are playing, must be in windowed mode with border. To note, you can use VLC Media Player or Windows Media Player to have a video display instead of a game in the overlay. My source codes and program are here.
